# Changelog

All notable changes to `lw` are documented here. Format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/); the project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

Per-release GitHub release notes are auto-generated by `goreleaser`
from conventional commits (`feat:` / `fix:` / `chore:` / etc.). This
file is the human-curated complement — what changed, why, and how it
affects users — for releases that move the needle.

## [Unreleased]

### Quality (Gruntwork-harden mission)

- **golangci-lint v2.5.0 ruleset adopted** with `--new-from-merge-base`
  ratchet. 44 linters from the gruntwork-io/terragrunt reference
  config; existing violations baselined into
  `internal/audits/golangci-baseline.json` (2612 issues tombstoned at
  adoption time).
- **codespell** + **dependabot** wired (`PR #42`).
- **`lw check` discipline hooks** enforced at pre-commit
  (companion-test + linked-incident requirements per CLAUDE.md §1, §8;
  closes lightwave-platform#633).
- **`.mise.toml`** pins Go, golangci-lint, goreleaser, go-junit-report
  versions across local dev + CI.
- **`go mod tidy` check** and **JUnit XML test reports** added to CI.
- **`internal/testutil` package** + testify integration — `NewPool`,
  `MakeEpic/Sprint/Story`, `RunHandler` helpers. CLAUDE.md gains a
  Test Conventions section.
- **CI restructured to reusable workflows.** `ci.yml` is now a thin
  orchestrator; lint, base-test, build, codespell, tidy-check, and
  pre-commit live in their own `workflow_call` modules. OS matrix
  (`ubuntu` + `macos`) for tests; cross-compile matrix (`linux,darwin
  × amd64,arm64`) for builds. Release pipeline reuses the same
  modules so what CI runs on `main` is what ships.
- **macOS notarization infrastructure** wired (`.gon_*.hcl` configs,
  `sign-macos.yml` reusable workflow, `docs/release-notarization.md`
  runbook). Dormant until Apple Developer secrets are provisioned —
  no functional change to tagged releases yet.

### Reverts

- Reverted `#49` (gomod-patch-minor group bump with bad rebase
  producing duplicate `require` blocks) and `#50` (tablewriter
  0.0.5 → 1.1.4 major API break that left 14 source files
  uncompilable). Dependabot disabled while we build a local-first
  replacement (`lw deps audit`, planned).
